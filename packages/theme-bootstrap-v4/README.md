# @vuecs/theme-bootstrap-v4

[![npm version](https://badge.fury.io/js/@vuecs%2Ftheme-bootstrap-v4.svg)](https://badge.fury.io/js/@vuecs%2Ftheme-bootstrap-v4)
[![main](https://github.com/Tada5hi/vuecs/actions/workflows/main.yml/badge.svg)](https://github.com/Tada5hi/vuecs/actions/workflows/main.yml)

Bootstrap 4.x theme for vuecs components. Includes an optional `--bs-*` → `--vc-color-*` bridge for consumer-authored CSS (Bootstrap 4's own components are Sass-compiled, see docs for the caveat).

Full documentation: **[vuecs.dev/themes/bootstrap-v4](https://vuecs.dev/themes/bootstrap-v4)**

```bash
npm install @vuecs/theme-bootstrap-v4 bootstrap@4
```
