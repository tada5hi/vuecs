# @vuecs/navigation

[![npm version](https://badge.fury.io/js/@vuecs%2Fnavigation.svg)](https://badge.fury.io/js/@vuecs%2Fnavigation)
[![main](https://github.com/Tada5hi/vuecs/actions/workflows/main.yml/badge.svg)](https://github.com/Tada5hi/vuecs/actions/workflows/main.yml)

Multi-level navigation for Vue 3 with `vue-router` integration and a shared NavigationManager for cross-component state. Also ships `<VCStepper>` (built on Reka's `StepperRoot`) for multi-step wizards / checkout / onboarding flows — owns its own state, no NavigationManager dependency.

Full documentation:

- [Navigation component](https://vuecs.dev/components/navigation)
- [Navigation Manager guide](https://vuecs.dev/guide/navigation-manager)
- [Stepper component](https://vuecs.dev/components/stepper)

```bash
npm install @vuecs/navigation
```
