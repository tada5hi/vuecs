# @vuecs/list

[![npm version](https://badge.fury.io/js/@vuecs%2Flist.svg)](https://badge.fury.io/js/@vuecs%2Flist)
[![main](https://github.com/Tada5hi/vuecs/actions/workflows/main.yml/badge.svg)](https://github.com/Tada5hi/vuecs/actions/workflows/main.yml)

Compound list components for Vue 3: a stateful root that provides context and thin part components (`Header`, `Body`, `Item`, `ItemText`, `ItemActions`, `Footer`, `Loading`, `Empty`) that read from it. Plus a `defineList()` factory + child-side `useList()` injector for shared state across views. Successor to `@vuecs/list-controls` — clean break, compound API.

Full documentation: **[vuecs.dev/components/list](https://vuecs.dev/components/list)**

```bash
npm install @vuecs/list
```
