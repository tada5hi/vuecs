# @vuecs/overlays

Overlay components for vuecs — Modal, Popover, Tooltip, DropdownMenu, ContextMenu — built as thin wrappers over [Reka UI](https://reka-ui.com/) primitives.

## Installation

```bash
npm install @vuecs/overlays @vuecs/core reka-ui
```

## Usage

See the [vuecs.dev documentation](https://vuecs.dev/components/modal) for full API reference, theme keys, and examples.
