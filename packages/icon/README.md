# @vuecs/icon

Iconify-backed icon component for vuecs. Wraps [`@iconify/vue`](https://iconify.design/docs/icon-components/vue/)'s `<Icon>` with a vuecs-namespaced `<VCIcon>`.

```bash
npm install @vuecs/icon
```

```vue
<VCIcon name="lucide:chevron-left" />
```

See [vuecs.dev/components/icon](https://vuecs.dev/components/icon) for the full reference, including how to wire icon delivery in Nuxt, Vite SSR, and Vite SPA setups.
